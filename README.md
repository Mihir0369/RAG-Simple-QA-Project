# Simple Q&A System
This repository contains a simple Retrieval-Augmented Generation (RAG) question-and-answer project.

## Project structure
- `app/` – application source code
- `requirements.txt` – Python dependencies

## Setup
1. Create and activate a virtual environment.
2. Install dependencies:
   - `pip install -r requirements.txt`

## Run
Run the main application entry point from the `app/` directory (or your configured start script).
